GameAutomation

## 環境構築

### 実行コマンド一覧


#### 仮想環境の構築
```
conda create -n GameAutoEnv python=3.7.5
```

#### 仮想環境に入る
```
conda activate GameAutoEnv
```

#### パッケージのアップデート
```
python -m pip install --upgrade pip
```

#### 必要パッケージのインストール
```
pip install pyautogui
pip install tqdm
pip install opencv_python
```
